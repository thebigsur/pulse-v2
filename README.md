# The Pulse v2

**LinkedIn Client Acquisition Engine for Financial Advisors**

Converts LinkedIn engagement into client conversations within ~2 hours/week.

## Quick Start

See `IMPLEMENTATION-GUIDE.md` for complete setup instructions.

## Stack

- **Frontend**: Next.js + React (Vercel)
- **Database**: Supabase (PostgreSQL)
- **AI**: Claude API (Haiku for scoring, Sonnet for generation)
- **Scraping**: Apify (LinkedIn, Twitter/X, TikTok)
- **Cost**: ~$75-110/month
